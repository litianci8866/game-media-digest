# 日报输出模板

## 标题格式

Game Media Digest - {YYYY年MM月DD日}

---

## 整体结构

——国内行业动态——

{游戏葡萄、游戏陀螺、手游那点事、游戏新知的内容，按媒体分组}

——出海动态——

{白鲸出海、Sensor Tower中国、伽马数据的内容，按媒体分组}

——海外行业动态——

{GamesIndustry.biz、Pocket Gamer、IGN的内容，按媒体分组；标题和要点翻译成中文}

---

## 每家媒体格式

【{媒体名}】

1. {文章标题（保留核心信息，去除营销语气；海外媒体翻译成中文）}
   · {关键点1（优先列数据或结论）}
   · {关键点2}
   · {关键点3（可选）}
   🔗 {原文链接}

2. ...（最多3篇）

---

## 格式规范

- 标题和板块名使用加粗（**文字**）
- 子弹点用「·」
- 每家媒体最多3篇，按重要性排序
- 每篇必须有原文链接，无链接不收录
- 无内容的媒体直接跳过，不写"暂无"
- 三个板块都无内容时，不生成日报
- 海外媒体文章标题和要点须翻译成中文，原文链接保留英文URL

---

## 结尾固定格式

数据来源：游戏葡萄 / 游戏陀螺 / 手游那点事 / 游戏新知 / 白鲸出海 / Sensor Tower中国 / 伽马数据 / GamesIndustry.biz / Pocket Gamer / IGN
